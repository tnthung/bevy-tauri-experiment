# Start the project

## 1. Install the dependencies

`pnpm i`

## 2. Build icons

`pnpm tauri icon ./src-tauri/icons/icon.png`

## 3. Run / Build the project

### Run the project in development mode

`pnpm tauri dev`

### Run the project in production mode

`pnpm tauri preview`

### Build the project

`pnpm tauri build`
